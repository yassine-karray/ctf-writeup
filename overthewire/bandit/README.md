# OverTheWire: Bandit

**Completed:** Levels 0–33  
**Platform:** [overthewire.org/wargames/bandit](https://overthewire.org/wargames/bandit)  
**Difficulty:** Beginner  
**Focus:** Linux fundamentals, SSH, file permissions, privilege escalation, git forensics

## What this wargame covers

Bandit is the entry point for OverTheWire. Each level teaches a real Linux concept  not theoretical, you learn by doing. By the end you have solid command-line fundamentals and a real feel for how privilege and access work on Linux systems.

## Key concepts learned

- SSH authentication (password + private key)
- Linux file system navigation and special filenames
- File permissions, setuid binaries, privilege escalation
- Shell escaping (`more`, `vi`)
- Special shell variables (`$0`)
- Networking basics (`ncat`, `openssl`, port scanning with `nmap`)
- Git forensics (log, branches, tags, pushing)
- Cronjob analysis
